# Open Financial System

An open, merit-based financial ecosystem — built to empower every human being with advanced tools to take control of their finances, at near-zero cost.

## Mission

> Maintaining and improving human life at near-zero cost.

The current global financial system is power-based. A merit-based financial system is a better solution. Crypto enables competition and merit. With 10,000+ blockchain ecosystems — each with their own wallets, DEXs, and bridges — it's not practical for general users to navigate them all. This project solves that problem.

## Projects

### [Open Wallet](open-wallet/)
The universal crypto wallet — one app for every token, every chain, every exchange. Swap BTC, ETH, SOL, ADA, and any token to stablecoins without leaving the wallet. Built-in DEX aggregation, cross-chain bridges, post-quantum encryption, bank account integration, and hardware wallet support. Designed for a farmer in rural India and a Wall Street analyst alike.

- **Mobile-first** — adaptive Simple Mode and Pro Mode
- **Platforms:** iOS, Android, Desktop
- **Tech:** React Native, Rust core, PQC (ML-KEM-1024, ML-DSA, SPHINCS+)

### [Open Chain](open-chain/)
A sovereign, mobile-first, post-quantum-ready blockchain. Built on Cosmos SDK + CometBFT with IBC interoperability to 50+ chains. Geographic sharding and sovereign rollups for planet-level scalability.

- **Progressive mobile-first:** server validators today, phone validators tomorrow, 100% P2P by ~2035
- **PQC native:** hybrid ML-DSA + Ed25519 on-chain signatures
- **Near-zero fees:** fixed cost per transaction, not percentage-based

### [Open Token](open-token/)
The native utility token of Open Chain. Fair launch — no pre-mine, no VC allocation. Value anchored to real usage as the gas token for all swaps and bridges. Governed by on-chain DAO.

## Principles

- **Open source** — 100% transparent, anyone can inspect and build from source
- **User in control** — self-custody, no intermediaries, your keys your coins
- **Planet-level scalability** — designed for 8 billion humans
- **Post-quantum security** — quantum-resistant from day one
- **Mobile-first** — design for mobile, build with best available, migrate to phone-only as tech catches up
- **Merit-based** — value from utility, not speculation or power

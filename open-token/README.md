# Open Token (OPEN)

The native utility token of Open Chain — designed for welfare, not speculation.

## Economics
- **Supply:** Capped (e.g., 21 billion OPEN)
- **Distribution:** 100% fair launch — no pre-mine, no VC allocation, no team reserve
- **Utility:** Gas token for all swaps/bridges within Open Wallet and Open Chain
- **Fees:** Near-zero fixed fees (~0.001 OPEN per transaction)
- **Inflation:** Low staking rewards (~3-5% annual) for validators
- **Burn:** Small % of swap fees burned — deflationary pressure tied to real usage
- **Value:** Usage-driven — more swaps = more demand for OPEN as gas = organic value

## Governance
- 1 OPEN = 1 vote
- On-chain proposals for fee changes, module upgrades, token burns, mobile migration decisions
